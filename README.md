# Sales Performance Analysis using SQL Server

## Project Overview

This project analyzes sales data using SQL Server to uncover valuable business insights. It demonstrates how SQL can be used to transform raw transactional data into meaningful reports for business decision-making.

The analysis includes sales trends, customer behavior, product performance, KPI reporting, and time-based analysis using advanced SQL queries.

---

## Objectives

- Analyze overall sales performance
- Identify top and bottom performing products
- Evaluate customer purchasing behavior
- Track monthly and yearly sales trends
- Generate key business KPIs
- Perform cumulative and comparative sales analysis
